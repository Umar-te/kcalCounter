# kcalCounter

A Website Inspired by Lifesum, to calculate food's macros and calories with user's specefic needs and daily intake 
Done using Nodejs, Expressjs and Mongoose(MongoDB). Incorperating RESTful APIs and MVC Model

By Eng/Umar Yasser 

Features: Signing Up and Logging In Users,
Setting Up Diet according to mifflin equation,
Tracking in a daily tracker
And of course adding foods and exercise for each daily tracker
All while maintaining best practices  for authentication & authorization such like using "helmet" package
